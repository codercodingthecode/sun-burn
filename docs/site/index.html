sun-burn reads a manifest from the image's boot partition and shows a setup wizard before flashing — WiFi, hostname, SSH, timezone. No monitor required.

Download for macOS View on GitHub

How it works

1

Pick an image

Select any .img file. sun-burn checks for a sunburn.json on the boot partition.

2

Configure

If a manifest is found, a guided wizard collects WiFi credentials, hostname, SSH keys — whatever the image author declared.

3

Flash

sun-burn writes the config to the boot partition, then flashes the image to your SD card or USB drive with real-time progress.

The sunburn.json standard

Any image author can ship a manifest. sun-burn renders the wizard from it.

sunburn.json 
{
  "version": "1",
  "name": "My Device OS",
  "steps": [
    {
      "id": "network",
      "title": "Network Setup",
      "fields": [
        { "id": "ssid", "type": "wifi-picker", "label": "WiFi Network", "required": true },
        { "id": "password", "type": "password", "label": "Password", "required": true }
      ],
      "writes": [
        { "path": "wifi.txt", "template": "ssid={{ssid}}\npassword={{password}}" }
      ]
    }
  ]
}

Field types

All eight types supported in the current spec.

Type Description
text Plain single-line text input.
password Masked text input.
wifi-picker Native or scanned Wi-Fi SSID chooser.
ssh-key-picker File-picker or paste for an SSH public key.
country-picker Dropdown seeded from the ISO 3166-1 country list.
timezone-picker Dropdown seeded from the IANA timezone database.
toggle Boolean on/off switch; value is "true" or "false".
select Dropdown from a static list; requires an options array.
